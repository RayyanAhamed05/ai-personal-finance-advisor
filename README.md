## 📌 Project Overview

**Transforming Personal Finance with AI Solutions** is an AI-powered personal finance management system designed to help users understand their financial habits, forecast future expenses, detect unusual spending, and receive intelligent financial recommendations.

The project integrates Machine Learning techniques with an interactive Streamlit dashboard, making financial analysis simple and accessible.

---

## 🎯 Objectives

- Help users monitor daily expenses.
- Predict future financial trends.
- Detect unusual transactions.
- Recommend personalized budgets.
- Improve financial planning.
- Encourage better saving habits.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/ai-personal-finance-advisor.git
```

### Navigate into the Project

```bash
cd ai-personal-finance-advisor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python -m streamlit run app/dashboard.py
```
